# Using Attenda on Windows

To use Attenda as a standalone application on Windows (One-Tab Setup), follow these steps:

## 1. Quick Launch
Run the `start_app.bat` file in the project folder. 
This will:
- Start the application server.
- Open the application in your default browser automatically.

## 2. Install as "App" (Recommended)
For the best experience (no address bar, dedicated window, start menu icon):

1.  Run `start_app.bat`.
2.  In **Chrome** or **Edge**:
    - Look for an "Install App" icon in the address bar (usually on the right side).
    - OR click the browser menu (⋮ or ...), go to **Apps**, and select **Install Attenda**.
3.  Once installed, you can launch Attenda directly from your Windows **Start Menu** or **Taskbar** without opening the browser first. Note that `start_app.bat` (server) needs to be running in the background for it to work.
